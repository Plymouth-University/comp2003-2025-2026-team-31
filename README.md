# COMP2003-2023
artofest.com – Festival Guide and Planner
=======================================

Project Overview
----------------
artofest.com is a cross-platform festival discovery and planning platform designed to help users explore, compare, and plan music festival experiences across Europe. The platform aims to centralise festival information and provide planning tools such as wish lists, filters, and a personalised planner.

The project is developed as part of the COMP2003 module and follows an agile, sprint-based project management approach.


Project Objectives
------------------
- Develop a responsive web platform for discovering European music festivals
- Provide detailed festival profiles (location, genre, dates, cost, media, booking links)
- Implement search, filtering, and planning features
- Design the system to support future mobile app and API integrations
- Apply software engineering best practices in planning, documentation, and collaboration

Development Approach
--------------------
The project follows an agile methodology organised into five sprints. Tasks are tracked using Trello, and progress is reviewed regularly through team meetings, client updates, and supervisor feedback.

Documentation, planning, and design were prioritised to ensure a scalable and maintainable solution.


Current Status

This repository currently contains project documentation, planning artefacts, and prototype-related materials. Core development is planned as the next phase following approval of the design and project plan.
